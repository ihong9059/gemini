# 콘텐츠 작성하기

Docsify의 모든 콘텐츠는 마크다운(`.md`) 파일로 작성됩니다.

## 기본 구조

일반적인 마크다운 문법을 그대로 사용할 수 있습니다.

```markdown
# 이것은 제목입니다

이것은 본문입니다. **굵은 글씨**와 *기울임 글씨*를 사용할 수 있습니다.

- 목록 1
- 목록 2

[이것은 링크입니다](https://docsify.js.org)
```

## 코드 블록

프로그래밍 코드를 삽입할 때는 코드 블록을 사용하고, 언어를 명시하면 문법 하이라이팅이 적용됩니다.

`
`
`bash
npm install
`
`
`

`
`
`javascript
console.log("Hello, Docsify!");
`
`
`
(참고: 위 코드 블록의 ` 와 ` 사이에 있는 공백은 설명을 위해 추가된 것으로, 실제 사용 시에는 붙여 써야 합니다.)
